# AI Based Crop Rotation Planning

AI Based Crop Rotation Planning is a project that utilizes artificial intelligence to optimize crop rotation strategies for farmers using stable matching and Gale-Shapley algorithm. The goal is to enhance soil health, increase crop yields, and promote sustainable farming practices.

### Structure of the project

This repository contains the following files:

- `CropRotation_Report.pdf`: A comprehensive report detailing the methodologies, results, and conclusions of the project.
- `crop_rotation.ipynb`: A Jupyter Notebook containing the code and analysis used in the project.
- `data/`: A directory containing the datasets used for testing the algorithm.
